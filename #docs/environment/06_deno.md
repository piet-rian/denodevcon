# Deno

* Denoを使えるようにするための作業は不要
  * devcontainer側の設定でdenoが最初からインストールされるようになっている

## 動作確認

```shell
##### devcontainer内で実行 #####
## denoの動作確認
deno -v

## サブコマンドやオプションの確認
deno -h
```

## Deno って何

* javascriptの実行環境
* 発音はディーノ
  * デノではない

## nodeじゃダメなの

* nodeの周辺が込み入っているのを嫌っている
  * フォーマッタ、リンタ、バンドラ、テスト等
  * Denoの場合はそれらを内蔵しているので、Deno単体で完結する
* Denoそのものがシングルバイナリ(≒コンテナネイティブ)
* Typescriptネイティブ
